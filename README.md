# Allfactive
Allfactive — A premium fragrance e-commerce platform showcasing scents from around the world. Developed with React and TypeScript, focusing on high-performance data fetching and a seamless user experience.


#Concept: 
O experiență digitală dedicată pasionaților de parfumuri, unde utilizatorii pot explora esențe rare și branduri internaționale.


#Tech Stack: 
React, TypeScript, Vite, TanStack Query (pentru gestionarea stării serverului), React Router (navigare), Context API (coș de cumpărături și autentificare).
